1. CHeck the status of apache server in all the app servers.
sudo systemctl status httpd

2. Check the port on which apache server is listening .
sudo ss -tulnp | grep httpd

3. Install Nginx in LBR server.
sudo yum install nginx -you
sudo systemctl start nginx 
sudo systemctl enable nginx
sudo systemctl status nginx 

4. Take a backup of nginx conf file .
mv /etc/nginx/nginx.conf /etc/nginx/nginx.conf.backup

5. Configure the nginx conf file with app server details .
 apapche server i slistening on port 8089 . add this below blck in nginx.conf file.

 http {
    # ... other settings ...
    
    # Upstream backend servers - USE PORT 8089
    upstream backend {
        server stapp01:8089;
        server stapp02:8089;
        server stapp03:8089;
    }

    server {
        listen       80 default_server;
        listen       [::]:80 default_server;
        server_name  _;

        location / {
            proxy_pass http://backend;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;
        }
    }
}

6. Test configuration
nginx -t

7. If successful, restart nginx
systemctl restart nginx

8. Verify it's running
systemctl status nginx

9. Check that nginx is listening on port 80
ss -tulpn | grep :80

10. Validate :
# From LBR server
curl -I http://localhost

# From jump host
curl http://stlb01

# Should return HTTP 200 OK
